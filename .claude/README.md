# Claude Code Configuration

Эта папка содержит конфигурацию для Claude Code AI assistant.

## Файлы

### config.json
Основная конфигурация проекта:
- Информация о проекте
- Технологический стек
- Настройки разработки
- Архитектурные решения

### prompts.md
Правила и руководства для разработки:
- Код-стайл и best practices
- Архитектурные паттерны
- Примеры частых задач
- Mobile-first и PWA требования

## Использование

Claude Code автоматически читает эти файлы и использует их как контекст при работе с проектом. Это помогает:

1. Поддерживать консистентность кода
2. Следовать установленным архитектурным решениям
3. Применять правильные паттерны
4. Соблюдать требования проекта

## Обновление

Обновляйте эти файлы по мере развития проекта:
- Новые архитектурные решения
- Изменения в стеке
- Новые правила и паттерны
- Важные соглашения команды
